# Simple Calculator

A lightweight **browser-based calculator** built with **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations and demonstrates fundamental **DOM manipulation**, **event handling**, and **state management** in JavaScript.

---

## Features

- Basic arithmetic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`×`)
  - Division (`÷`)
- Real-time input display
- Clear (`C`) button to reset the calculator
- Responsive button interface

---

## Technologies Used

| Technology | Purpose                                            |
| ---------- | -------------------------------------------------- |
| HTML       | Provides the structure of the calculator interface |
| CSS        | Styles the layout and visual appearance            |
| JavaScript | Implements calculator logic and interactivity      |

---

## How It Works

1. The user presses number buttons to input values.
2. Operator buttons store the selected operation.
3. When `=` is pressed, the calculator evaluates the expression.
4. The result replaces the display value.
5. Pressing `C` clears the current input and resets the calculator state.
